ABSTRACT
This paper investigates cold start acquisition schemes for
a GPS L5 software receiver. Since the chipping rate of L5
PRN code is 10.23 MHz, which is 10 times that of the L1
C/A-code, one major concern for the software signal
processing is the computation load. The proposed
algorithm should have a reasonable computation burden
in order to be implemented into a software receiver.
In particular, two new methods are proposed. The main
idea of the first approach is to eliminate the frequency
information of the incoming signal, thus only one
dimension search of the initial point of the PRN code is
required in the first step. The acquisition of the Doppler
frequency and NH code can be easily fulfilled after the
detection of the chip offset. When the signal strength is
normal or above, the computation burden of this method
is very low, which makes it well suited for a software
signal acquisition. The second approach can detect the L5
signal in a very low signal-to-noise ratio, such as indoors.
Longer coherent integration is expected using Q5
correlator outputs for a higher correlation gain. A
combined L5 codes and Neumann Hoffman (NH) codes
acquisition algorithm is proposed by Hegarty et al. [2003]
using very long coherent integration times (i.e. 20 ms or
longer) on dataless channel. However, this method is not
practical for software signal processing due to its heavy
computation load. A shortened fast Fourier transform
(FFT) version of the above method, which is suitable for
the software receiver, can greatly reduce the computation
burden without degradation on performance. In case of a
weak signal, the coherent integration time can be
extended for a very long period to get enough correlation
gain. The only limitation is the quality of the receiver’s
oscillator.
INTRODUCTION
This paper presents two new cold acquisition approaches
suitable for a GPS L5 software receiver. The first method
(Frequency Removal method) multiplies the received
Intermediate Frequency (IF) samples with its slightly
delayed version. The outcome of this multiplication is a
“new code” modulated on two carrier components: a dc
term and a high frequency term. The “new code” is the
product of a PRN code and its delayed version.
Simulations show that the autocorrelation of this “new
code” can be used to find its initial point, which is the
same as that of the PRN code. The high frequency term of
the modulated carrier can be easily filtered out using a
pre-detection filter (accumulate and dump). The delay of
the IF samples must be carefully chosen in order to make
the dc term close to one. An advantage of this algorithm is
that the NH code and data modulations can be mostly
removed by multiplication. As a result, longer coherent
integration can be achieved. This method can apply
correlation with the data (I5) and/or dataless (Q5)
components of the incoming L5 signal. It is found that the
correlation with the Q5 channel slightly outperforms that
with the I5 channel, which will be discussed later in the
paper.
The second algorithm (NH Removal method) searches
three unknowns - code delay, Doppler frequency, NH
code initial point - on the dataless channel. The start point
of the following tracking process can be found by the

detection of the highest correlation peak above the
threshold. It is a shortened FFT version of an acquisition
algorithm proposed by Hegarty et al. [2003], which is
suitable for weak signal acquisition. When a software
receiver is adopted to implement Hegarty’s algorithm,
where the FFT method is normally used to compute the
correlation values, the computation load is so heavy that it
makes it impractical. The second acquisition algorithm
proposed here can greatly reduce the computation load
and is suitable for software signal processing. In the case
when a 20 ms coherent integration time is implemented,
assuming a sampling rate of 42.16 MHz (42,160 points
FFT for 1 ms), instead of carrying out twenty 42,160-
point FFT, only one 42,160-point FFT has to be
performed. When longer coherent integration (i.e. > 20
ms) is performed to get a higher correlation gain, the
computation load may be reduced further depending on
the line-of-sight dynamics and the quality of the
receiver’s oscillator.
The performance comparison of the algorithms mentioned
above, as well as the non-coherent integration scheme, is
described in the third part. Finally, this paper concludes
with a short summary.
SCHEME 1 - FREQUENCY REMOVAL (FR)
This method multiplies the incoming signal with its
slightly delayed version to generate a new PRN code. The
code delay of this new code is found to be the same as
that of the Q5 PRN code that can be used to find the
initial phase of the incoming Q5 code. The modulated
carrier and NH codes can be mostly eliminated by
multiplication and further filtering. Only one dimension
search of the code delay has to be performed at the first
step, which makes this method very fast. Once the initial
phase of the Q5 code is found, the Doppler frequency and
NH code acquisition can be easily fulfilled. Previous
implementation of this method on L1 C/A-code
acquisition can be found in [Tsui, 2000] and [Lin & Tsui,
2000].

For 1-second coherent integration, the code delay shift
can be as high as 43.5 chips. In this case, no correlation
peak can be detected by adding the correlation values at

A code Doppler searching method is proposed here to
figure out this problem. For 1-second coherent
integration, the code Doppler searching range is set to –
43.5 Hz to 43.5 Hz with 1-Hz step sizes. The correlation
values of each 1-ms period will be shifted according to
the code Doppler and then summed together. The code
Doppler can be found by the detection of the highest peak
of the accumulated correlation value. Using this method,
the code delay and carrier Doppler can be acquired at the
same time, which further speeds up the acquisition

SCHEME 2 - NH REMOVAL (NHR)
Although scheme 1 discussed above can acquire the
signal very fast, it may not be suitable to detect weak
signals because of the increase in noise power. In order to
acquire weak signals, an acquisition algorithm proposed
by Hegarty et al. [2003] is implemented and investigated.
Single or multiple 20-ms coherent integration periods on a
dataless channel is adopted in this method, which makes it
capable of acquiring weak signals. However, since there
are three unknowns (code delay, Doppler frequency, NH
code initial point) to search, the computation load of this
method is so heavy that it makes it impractical for
software signal processing. A shortened FFT version of
this method is proposed here that can greatly reduce the
computation burden.
The main idea is to divide the 20-ms samples equally into
20 sections and add them together to create 1-ms long
samples. These newly generated 1-ms samples are used to
detect the correlation peak. Instead of carrying out 20-ms
FFT, only a 1-ms FFT has to be performed. Figure 8
shows the acquisition result when the Doppler bin size is
set to 666.7 Hz. It is apparent that no correlation peak has
been found. This is due to the 20-bit NH code modulation.
The autocorrelation property of this short code is highly
degraded when the difference between the true Doppler
and the estimated Doppler is large.
Figure 8: Acquisition result, step size = 666.7 Hz
In order to detect the correct correlation peak, the Doppler
bin size has to be set to a reasonable small value. Figure 9
shows the acquisition result with a 33.3-Hz step size. In
this case, the correlation peak has been detected
successfully. Actually, a fine frequency search has to be
carried out to avoid false acquisition of the NH code. A
detailed discussion of this issue can be found in
[Macabiau et al. 2003].

CONCLUSIONS
Two new acquisition schemes for a GPS L5 software
receiver have been described. Simulation results show
that they are suitable for software signal processing under
different scenarios. When the signal level is above normal
(C/N0 >= 46 dB-Hz), the frequency removal method can
detect the signal with the lowest computation load. When
the signal level is below normal (C/N0 < 46 dB-Hz), the
Neumann-Hoffman removal method or the non-coherent
integration method is preferred to obtain enough
correlation gain.
